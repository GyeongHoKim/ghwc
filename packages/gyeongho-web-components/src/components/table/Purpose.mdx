# 요구사항

- 테이블 컴포넌트는 데이터에 종속적이지 않아야 합니다.
- 테이블 컴포넌트의 각 셀 내부에 또다른 컴포넌트가 들어갈 수 있어야 합니다.
- 스크롤과 페이지네이션 기능이 있고 그걸 선택할 수 있어야 합니다.
  - 스크롤 시 헤더가 고정되어야 합니다.
- 정렬기능과 필터 기능이 있고 해당 로직을 일급 함수로 받아야 합니다.
- 테이블 컴포넌트의 각 열의 너비를 조절할 수 있어야 합니다
  - 이때, 픽셀단위나 비율단위 둘 다 가능해야 합니다(table이 아니라 header cell에서 받아야 함)
  - 런타임동안에 마우스 드래그를 통해서도 조절 가능해야 합니다.
    - text의 크기보다 줄어들었을 때 ...으로 할지 불가능하게 할지도 선택할 수 있어야 합니다(prop을 통해서든 css를 수정가능하게 part를 만들든)
- 테이블 컴포넌트 자체의 너비와 높이가 고정이어야 합니다.
  - 명시하지 않을 때는 항상 상위 컴포넌트의 100%를 차지해야 합니다.
- part와 css변수가 자세해야 합니다.
  - border, font, line height 등
